## Comandos de Infraestructura Frontend

- **Full Update Astro:** 1. `npx taze -u` (Actualiza el package.json a las últimas versiones compatibles).
  2. `npm install` (Instala las nuevas versiones).
  3. `npm audit fix` (Resuelve problemas de seguridad).
  4. `npx astro check` (Verifica errores de TypeScript y sintaxis de Astro).
  5. `npm run build` (Verificación final de integridad).
